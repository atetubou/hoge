# hoge

## How to setup

```
npm install
```

## How to run

For api serving.

```
go run ./server/main.go
```

For html/javascript serving.

```
npm run serve
```

## misc

Apply formatter using [deno](https://deno.land/#installation).

```
deno fmt $(git ls-files)
```
